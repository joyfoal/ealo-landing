# 이뤄 (Ealo) — 랜딩 페이지

AI 시대의 불안 속에서 40·50대가 매일 성공의 말(확언)을 카메라를 보며 소리 내어 말하는 앱, **이뤄**의 소개 랜딩 페이지입니다.

## 구성
- `index.html` — **웹(데스크톱) 랜딩 페이지**
- `mobile.html` — **모바일 랜딩 페이지** (휴대폰 화면에 맞춘 세로 레이아웃)
- `support.js` — 페이지 렌더 런타임 (두 HTML이 사용)
- `assets/` — 로고(`ealo-logo.png`), 카메라 말하기 영상(`man-speak.mp4`, `woman-speak.mp4`)

> 폰트(Pretendard)는 CDN에서 불러오므로 인터넷 연결이 필요합니다.

## 로컬에서 보기
정적 서버로 열어주세요 (영상·스크립트가 상대 경로로 로드됩니다):
```
npx serve .
```
그런 다음 `http://localhost:3000/index.html` (웹) 또는 `/mobile.html` (모바일)을 엽니다.
※ 파일을 브라우저로 바로 더블클릭하면 일부 브라우저에서 스크립트·영상이 차단될 수 있어요.

## GitHub Pages로 배포
1. 이 폴더의 파일을 GitHub 저장소 루트에 올립니다.
2. **Settings → Pages → Source**에서 브랜치(`main`)와 `/(root)`를 선택합니다.
3. `https://<사용자명>.github.io/<저장소명>/` → 웹 랜딩, `.../mobile.html` → 모바일 랜딩.

## 카메라 말하기 데모
히어로의 카메라 화면은 실제 영상(`man-speak.mp4` → `woman-speak.mp4`)이 번갈아 재생되며,
"나는 / 매일 / 성장하고 / 있다" 단어가 순서대로 밝아지고, 끝나면 축하 화면이 나타납니다.

## 문구 원칙
인지심리학 근거자료에 따라 "말하면 이루어진다" 같은 단정 표현 대신,
"보고, 말하면, 더 오래 기억됩니다"처럼 연구·이론에 기반한 표현을 사용합니다.
이뤄는 목표 달성을 보장하지 않으며, 목표를 더 자주 인식·기억하도록 돕는 습관 형성을 지향합니다.
