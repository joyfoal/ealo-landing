# 이뤄 (Ealo) — 랜딩 페이지

확언("성공의 말")을 카메라를 보며 소리 내어 말하는 앱, **이뤄**의 소개 랜딩 페이지입니다.

## 데모 보기
`index.html`을 브라우저에서 열면 됩니다. 모든 리소스(폰트·이미지·스크립트)가 파일 하나에 인라인되어 있어 **인터넷 없이도** 그대로 열립니다.

## GitHub Pages로 배포
1. 이 저장소를 GitHub에 올립니다.
2. **Settings → Pages → Source**에서 브랜치(`main`)와 루트(`/`)를 선택합니다.
3. 잠시 후 `https://<사용자명>.github.io/<저장소명>/` 에서 랜딩 페이지가 열립니다.

`index.html`이 저장소 루트에 있으므로 별도 빌드 과정이 필요 없습니다.

## 파일
- `index.html` — 랜딩 페이지 (편집도 이 파일에서 바로 합니다)
