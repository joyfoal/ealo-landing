# CLAUDE.md — 이뤄 (Ealo) 랜딩 페이지

이 폴더는 정적 웹사이트입니다. **빌드 도구·번들러·npm 의존성이 없습니다.** 파일을 그대로 정적 호스팅(GitHub Pages 등)에 올리면 동작합니다.

## 파일 구조
- `index.html` — 데스크톱(웹) 랜딩 페이지 (본문 폭 1440px)
- `mobile.html` — 모바일 랜딩 페이지 (본문 폭 390px)
- `support.js` — 두 HTML이 공통으로 사용하는 렌더 런타임 (수정하지 마세요)
- `assets/` — `ealo-logo.png`(로고), `man-speak.mp4`·`woman-speak.mp4`(카메라 말하기 영상)

## 렌더 방식 (중요)
`index.html`·`mobile.html`은 일반 HTML이 아니라 **Design Component** 형식입니다.
- `<x-dc>` 태그 안의 마크업 + 하단 `<script data-dc-script>`의 `class Component extends DCLogic { ... }` 로직으로 구성됩니다.
- `support.js`가 이 둘을 읽어 화면을 그립니다.
- 템플릿의 `{{ 이름 }}`은 로직 클래스 `renderVals()`가 반환한 값으로 채워집니다.
- `<sc-for>`(반복), `<sc-if>`(조건)로 목록·분기를 표현합니다.
- **스타일은 전부 인라인 `style=""`** 로 작성합니다. (CSS 클래스/외부 스타일시트 미사용)

## 편집 가이드
- 텍스트·색상·간격 수정: `<x-dc>` 안의 해당 인라인 스타일/텍스트를 직접 고치세요.
- 카드 목록(근거·단계·기능 등)의 내용: 하단 로직 클래스 `renderVals()`의 `evidence`/`steps`/`features` 배열을 수정하세요.
- `index.html`을 고치면 `mobile.html`에도 같은 내용을 반영해야 합니다(두 파일은 독립적입니다).

## 카메라 말하기 데모 동작
폰 화면의 영상은 `man-speak.mp4` → `woman-speak.mp4`를 번갈아 **무한 반복**합니다.
재생 중 "나는 / 매일 / 성장하고 / 있다" 단어가 순서대로 밝아지고, 각 영상 끝에서 축하 화면이 잠깐 나타납니다.
화면 하단 원형 버튼으로 **일시정지/재생**할 수 있습니다.
- 단어 하이라이트 타이밍: 로직 클래스의 `thresholds` 배열(영상 진행률 기준)
- 자동재생 정책상 영상은 **음소거(muted)** 로 시작합니다.

## 로컬 실행
정적 서버로 여세요 (영상·스크립트가 상대 경로로 로드됨):
```
npx serve .
```
→ `http://localhost:3000/index.html` (웹) / `/mobile.html` (모바일)
※ 파일을 브라우저로 바로 더블클릭하면 일부 브라우저에서 스크립트·영상이 차단될 수 있습니다.

## 문구 원칙
인지심리학 근거자료에 따라 **"말하면 이루어진다" 같은 단정 표현은 사용하지 않습니다.**
"보고, 말하면, 더 오래 기억됩니다"처럼 연구·이론에 기반한 표현을 쓰고, 이뤄는 목표 달성을 보장하지 않으며 목표를 더 자주 인식·기억하도록 돕는 습관 형성을 지향합니다.
